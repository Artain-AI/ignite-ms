# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Send reports to:

```text
security@artain.com
```

Include enough detail to reproduce the issue, including affected commit or
version, environment, input files if relevant, and expected impact.

## Scope

Security reports may include issues in:

- Rust engine code
- native TensorRT bridge code
- Docker image configuration
- model/download/provisioning scripts
- benchmark and quickstart scripts

## Response

Artain will review security reports and respond as soon as practical. We may ask
for additional reproduction details before confirming impact or publishing a fix.

Please give us reasonable time to investigate before public disclosure.
